Full Retropie Guide

    Originally Written: 7/18/2024


    Retropie is an emulation operating system that can run on many systems but as it's name suggests, it was originally designed for the Raspbery Pi boards. I recently purchased a Raspberry Pi 5 (as of writting) and wanted to push it to it's absolute limits to play the most recent consoles that it "supports" including the Gamecube and the Wii. As such this guide will be focused on the Raspberry Pi 5 but i will be adding side comments for the Raspberry Pi 4 and will update this article for any future boards or updates to the operating system.

    This guide will cover how to push the Raspberry Pi's performance to be able to play the most recent consoles as best as possible. This process includes installing Retropie, overclocking, modifying the RAM split, installing emulators such as dolphin (Wii/Gamecube), accessing the hidden console and desktop enviornment, and using that enviornment to configure those emulators.


    If you have any suggestions or additions that should be added to this page, feel free to email me.


    How to install Retropie

    There are two ways to install Retropie. You can install retropie with the official Raspberry Pi Imager, or you can build Retropie on Raspberry Pi OS. However, I don't know if you can complete this full guide with the official image because as of writing there is no official image for the Raspberry Pi 5.

    Option 1: Install through the Raspberry Pi Imager

    1. Start by downloading the Raspberry Pi Imager, then launch the application.

    2. Select your Raspberry Pi device.

    3. Select the Retropie Operating System: Emulation OS > Retropie > Select the version for your pi. If your pi's version is not avalible you will have to build Retropie on Raspberry Pi OS.

    4. Select your Storage Device. You can boot off a micro sd card (recommended), usb drive, or through PCIe (Pi5 Only)

    5. Click on next, and edit settings.

      This opens the Operating System Customization Menu. This allows us to make it easier to install the operating system.

    6. Enable set hostname and change the host name to "retropie" this will help us identify our Raspberry Pi in the network easier.

    7. Enable set username and password and change them to your usual username and password. Please remember this password because you will need it later. I also recommend making the password short and easy to remember because this will be your admin password on the Raspberry Pi.

    8. Enable configure wireless LAN. This will autofill with the network you are currently on.

    9. Finally, enable set locale settings. This is what your menu should now look like:

    10. Select save, and then click yes. Once your storage device is imaged, connect it to your Raspberry Pi and turn the Pi on.

    11. Welcome to retropie!

    Option 2: Build on Raspberry Pi OS

    1. Start by downloading the Raspberry Pi Imager, then launch the application.

    2. Select your Raspberry Pi device.

    3. Select Raspberry Pi OS (64-bit) as the Operating System.

    4. Select your Storage Device. You can boot off a micro sd card (recommended), usb drive, or through PCIe (Pi5 Only)

    5. Click on next, and edit settings.

      This opens the Operating System Customization Menu. This allows us to make it easier to install the operating system.

    6. Enable set hostname and change the host name to "retropie" this will help us identify our Raspberry Pi in the network easier.

    7. Enable set username and password and change them to your usual username and password. Please remember this password because you will need it later. I also recommend making the password short and easy to remember because this will be your admin password on the Raspberry Pi.

    8. Enable configure wireless LAN. This will autofill with the network you are currently on.

    9. Finally, enable set locale settings. This is what your menu should now look like:

    10. Select save, and then click yes. Once your storage device is imaged, connect it to your Raspberry Pi and turn the Pi on.

    11. Open the terminal Application and Execute these lines:

      12. sudo apt install git lsb-release
      git clone --depth=1 'https://github.com/RetroPie/Retropie-Setup.git'
      cd Retropie-Setup
      chmod +x retropie_setup.sh
      sudo ./retropie_setup.sh

    12. This will install retropie and open the retropie setup script. To navigate this script use the arrow keys and select with enter. Select Basic Install and Install Retropie.

    13. Once this is complete go to the Configuration / Tools Menu and enable autostart and bashwelcometweak

    14. Now restart the raspberry pi, you can also use the command sudo reboot for this.

    15. Welcome to Retropie!
